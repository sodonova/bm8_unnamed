GeoMessenger

GeoMessenger is an app that allows for unique spatial messaging - you can leave a message in a bottle with a code at a location, and later anyone with the code can read it.

Send Messages to friend on the map

If your friend puts a message at a location, you can go by that location and then as long as you have the key, you can read the message! An interesting way to go about sending messages!

What the app is built off of

The main API used in the app is radar.io. This allows the app to get latitude and longitude coordinates. The pin marks on the map are stored using firebase. This allows you to have your pins that you made saved along with the message you put in!
